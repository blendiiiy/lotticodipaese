# 21.dev Components

Place components from 21.dev here.

## How to Add Components

1. Go to https://21st.dev/studio
2. Find a component you like
3. Copy the code
4. Create a new file here: `components/21dev/[component-name].tsx`
5. Paste the code
6. Import and use in your pages

## Component Categories

- **Hero Sections** - Eye-catching landing sections
- **Cards** - Service/product cards
- **Galleries** - Image showcases
- **Forms** - Contact/booking forms
- **Animations** - Interactive elements
- **Navigation** - Menu components
