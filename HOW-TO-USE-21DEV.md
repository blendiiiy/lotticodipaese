# How to Use 21.dev Components

## What is 21.dev?

21.dev (21st.dev) is a platform with over 6,000 ready-to-use React and Tailwind components. Perfect for building modern, beautiful websites!

## How to Get Components

### Method 1: Browse and Copy
1. Visit https://21st.dev/studio
2. Browse components by category
3. Click on a component you like
4. Copy the code
5. Paste into your project

### Method 2: Use Magic Component Platform (MCP)
1. Install the 21.dev MCP tool
2. Describe what component you need
3. It generates the code automatically

## Integration Steps

1. **Copy Component Code** from 21.dev
2. **Create Component File** in `components/21dev/`
3. **Import and Use** in your pages
4. **Customize** colors and styles to match your brand

## Example

```tsx
// components/21dev/animated-hero.tsx
// Paste component code from 21.dev here

// Then use it:
import { AnimatedHero } from "@/components/21dev/animated-hero"
```
