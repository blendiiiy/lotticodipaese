# Video Directory

Place your hero video here.

## Video Requirements

- **Format**: MP4 (recommended)
- **Resolution**: 1920x1080 or higher
- **File size**: Under 10MB (for best performance)
- **Duration**: 10-30 seconds (loops automatically)

## How to Add Video

1. Download the video from iStock or your source
2. Save it as: `public/videos/hero-video.mp4`
3. The video will automatically play in the hero section

## Video from iStock

If you purchased the video from iStock:
- Download the MP4 file
- Place it in this folder
- Name it: `hero-video.mp4`

## Alternative: Use Video URL

If you have a video URL, you can update `app/page.tsx` to use the URL directly instead of a local file.
